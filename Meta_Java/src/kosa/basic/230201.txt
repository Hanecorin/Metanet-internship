상수 - Constants

	final 키워드를 사용하여 상수를 표시합니다
	
	final로 선언된 변수는 값이 한번 할당되면 변경할 수 없습니다.
	
	final double PI = 3.14;
	System.out.println("PI is " + PI);


자료형 - Data Type

	자료형(타입)은 변수가 가지는 자료의 종류
	
		- 모든변수는 자료형을 갖습니다.
		- 자료형에 따라 할당하는 메모리 크기가 달라집니다.
	
	자바의 기본 자료형
		- 정수형 : int, short, long ,byte
		- 실수형 : float, double
		- 문자형 : char
		- 부울형 : boolean
		
		# boolean(부울형) 빼고는 데이터 변환이 모두 가능하다.
